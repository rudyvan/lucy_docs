# calendar_driver

<!--e_name-->

<!--s_role-->
<!--e_role-->

<!--s_descr-->
Needed for: 1)Notifications posted in a Things calendar or in notes, 2)Reading of triggers to start events, 3)Authorization Access time check from the calendar
